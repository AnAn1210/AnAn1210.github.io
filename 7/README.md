# 英文單字測驗

這個資料夾是可部署到 Google Apps Script 的選擇題測驗專案，介面做成你截圖那種卡片式版面，總分固定 100 分。

## 功能
- 從 Google 試算表隨機出題
- 每次出 10 題，答完立即顯示分數
- 選項採 4 選 1，會自動顯示對錯
- 若尚未設定試算表，會自動使用內建示範題庫

## 試算表格式
建議建立一個工作表，名稱用 `題庫`，系統會自動辨識常見欄位：
- 英文單字欄：`英文單字`、`單字`、`word`、`vocab`、`term`
- 中文意思欄：`中文意思`、`意思`、`meaning`、`answer`、`translation`、`解釋`、`翻譯`

最簡單的格式仍然是：
- A 欄：英文單字
- B 欄：中文意思

第一列可放標題，例如 `英文單字`、`中文意思`。

## 設定方式
1. 建立一個新的 Google Apps Script 專案。
2. 將 `7/Code.gs`、`7/index.html`、`7/appsscript.json` 內容貼到專案中。
3. 如果要改成自己的試算表，請到專案設定中新增 Script Property：
   - `QUIZ_SPREADSHEET_ID`：你的試算表 ID
   - `QUIZ_SHEET_NAME`：題庫工作表名稱，預設是 `題庫`
4. 重新整理 Web App，就能開始測驗。

## 備註
- 系統會依題庫內容自動組合錯誤選項
- 若題庫少於 10 題，仍可執行，但出題數會跟著減少
